# DM 一人回しツール

デュエルマスターズのデッキを一人で回して試せるツールです。

---

## すぐ使いたい人（おすすめ）

ブラウザで以下のURLを開くだけで使えます。インストール不要。

**https://MIYAMOTO5963.github.io/dm-solitaire/dm-solitaire-web.html**

Chrome か Edge で開いてください。

---

## 使い方

### 1. デッキを作る

1. 右上の「デッキ管理」を開く
2. 「新規作成」でデッキ名を入力
3. カード名で検索して、枚数を指定して追加
4. 40枚揃えたら完成

デッキデータはブラウザに自動保存されます。

### 2. ゲームを始める

1. 使うデッキを選んで「ゲーム開始」
2. シールドが5枚セットされて初期手札が配られます

### 3. 基本操作

| 操作 | 内容 |
|---|---|
| カードを左クリック | タップ / アンタップ |
| カードを右クリック | 移動・各種アクション |
| リボン上部のボタン | ドロー、ターン終了など |

---

## デッキを別のPCやスマホに移したい

1. デッキ管理画面の「書き出し」を押してJSONファイルを保存
2. 移行先で「読み込み」からそのファイルを開く

---

## ローカルで動かしたい人向け

ダウンロードして自分のPCで使うこともできます。

### 手順

1. このページの「Code」→「Download ZIP」でダウンロード
2. ZIPを解凍して `dm-solitaire.html` をブラウザで開く

カード検索も使いたい場合は Python が必要です。

```
python dm-proxy.py
```

を起動した状態で `dm-solitaire.html` を開いてください。

### Python のインストール（入っていない場合）

**Windows:**
```
winget install Python.Python.3
```

**Mac:**
```
brew install python
```

または [python.org](https://python.org) からインストーラーをダウンロード。
